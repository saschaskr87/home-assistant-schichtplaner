# Projektübersicht / Project Overview

🇩🇪

Dieses Projekt entstand aus dem Wunsch heraus, ein flexibles, mobiles und mehrbenutzerfähiges Schicht- und Haushaltsplanungssystem innerhalb von Home Assistant zu entwickeln.

Das System unterstützt:

* Mehrpersonenplanung
* Familienkalender
* mobile Dashboardsteuerung
* Statistiksysteme
* Mehrtagelogik
* automatische Kalenderverarbeitung
* langfristige Haushaltszyklen

🇬🇧

This project was created to provide a flexible, mobile and multi-user shift and household planning system within Home Assistant.

The system supports:

* multi-user planning
* family calendars
* mobile dashboard control
* statistics systems
* multi-day logic
* automated calendar processing
* long-term household cycles

---

# Features

* Mobile Touch Dashboard
* Multi-user shift planning
* Dynamic routing logic
* Google Calendar / CalDAV support
* Statistics engine
* Historical data support
* Reload protection
* Ghost booking prevention
* Family planner integration
* Multi-day booking logic
