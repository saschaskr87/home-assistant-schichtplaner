# Master Documentation
## Projektziel
Der Schichtplaner wurde entwickelt, um ein flexibles und mehrbenutzerfähiges
Planungs- und Statistiksystem für Home Assistant bereitzustellen.
## Hauptfunktionen
- Mehrpersonenplanung
- Kalenderintegration
- Statistiksystem
- Mehrtagelogik
- Mobile Bedienung
- Reloadschutz
## Architektur
Frontend Layer → Routing Scripts → Calendar Layer → Statistics Engine
## Wichtige Entwicklungsprobleme
- Geisterbuchungsproblem
- Reloadprobleme
- Timingprobleme
- Statistikinstabilität
- Kalender-Synchronisierung
